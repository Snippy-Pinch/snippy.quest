Gmail SMTP Setup Guide

Configure Gmail for sending emails with Clawdbot

Prerequisites

  • Gmail account (e.g., clawdbot@bainter.xyz)
  • 2-Factor Authentication enabled on the Gmail account

Setup Steps

1Generate Gmail App Password

  1. Go to Google Account Settings:
    Visit: https://myaccount.google.com/apppasswords
    Sign in with your Gmail account
  2. Enable 2-Factor Authentication (if not already enabled):
    Go to: https://myaccount.google.com/security
    Click "2-Step Verification" and follow the setup wizard
  3. Create App Password:
    • Go back to: https://myaccount.google.com/apppasswords
    • Select app: "Mail"
    • Select device: "Other (Custom name)"
    • Enter name: "Clawdbot SMTP"
    • Click "Generate"
    • Copy the 16-character password (looks like: abcd efgh ijkl mnop)
    • Remove spaces when storing: abcdefghijklmnop

2Store Gmail Credentials

Create the credentials file:

cat > /root/.clawdbot/secrets/gmail_smtp.json << 'EOF'
{
  "email": "clawdbot@bainter.xyz",
  "password": "abcdefghijklmnop",
  "smtp_server": "smtp.gmail.com",
  "smtp_port": 587,
  "from_name": "Snippy",
  "added": "2026-02-08",
  "notes": "Gmail SMTP for sending emails"
}
EOF

# Secure the file
chmod 600 /root/.clawdbot/secrets/gmail_smtp.json

Troubleshooting

"Username and Password not accepted"

  • Make sure 2FA is enabled on the Gmail account
  • App password must be 16 characters, no spaces
  • Try regenerating the app password

"SMTP AUTH extension not supported"

  • Make sure you're using port 587 (not 465)
  • Enable "Less secure app access" if app passwords don't work
  • Check that SMTP is enabled in Gmail settings

"Daily sending quota exceeded"

  • Gmail free accounts: 500 emails/day
  • Wait 24 hours or upgrade to Google Workspace

Security Notes

  • Never commit credentials to Git - they're in .gitignore
  • Use App Passwords - more secure than account password
  • Rotate passwords - regenerate app passwords periodically
  • Monitor usage - check Gmail "Sent" folder for unexpected activity
