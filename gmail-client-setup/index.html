Gmail "Send Mail As" Setup

Send emails from a custom address using your Gmail account

What This Does

Gmail's "Send mail as" feature lets you send emails that appear to come from a different email address (like a custom domain) while using your regular Gmail account.

Example: You have linux.trevor@gmail.com but want to send emails that show snippy@snippy.quest as the sender.

Prerequisites

  • A Gmail account (e.g., linux.trevor@gmail.com)
  • The custom email address you want to send from (e.g., snippy@snippy.quest)
  • Access to that custom email's inbox (to receive the verification email)

Setup Steps

1Open Gmail Settings

  1. Go to Gmail
  2. Click the gear icon ‚öôÔ∏è in the top right
  3. Click "See all settings"
  4. Go to the "Accounts and Import" tab

2Add Your Custom Email Address

  1. Find the section "Send mail as:"
  2. Click "Add another email address"
  3. A popup window will appear
  4. Enter:
    • Name: How you want your name to appear (e.g., "Snippy")
    • Email address: Your custom email (e.g., snippy@snippy.quest)
  5. Uncheck "Treat as an alias" if you want separate tracking
  6. Click "Next Step"

3SMTP Settings (Use Gmail's Servers)

Gmail will ask how to send mail. Use these settings:

  • SMTP Server: smtp.gmail.com
  • Port: 587
  • Username: Your Gmail address (e.g., linux.trevor@gmail.com)
  • Password: Your Gmail password (or App Password if 2FA is enabled)
  • Secured connection using TLS: Yes

  1. Gmail will send a verification email to snippy@snippy.quest

  4. Either:

Troubleshooting

Verification email not arriving

  • Check spam/junk folders
  • If you have email forwarding, check the forwarding destination
  • Wait 5-10 minutes (sometimes delayed)
  • Try resending the verification email from Gmail settings

Authentication error when sending

  • Make sure you're using your Gmail password (not the custom email's password)
  • If 2FA is enabled, use an App Password instead
  • Verify SMTP settings: smtp.gmail.com, port 587, TLS enabled

"Treat as an alias" option

  • Checked: Replies go to your Gmail inbox (simpler)
  • Unchecked: Replies go to the custom email (separate tracking)

How It Works

When you send an email using "Send mail as," Gmail sends the email through its own servers but sets the From: header to your custom email address. Recipients see the custom address as the sender.
